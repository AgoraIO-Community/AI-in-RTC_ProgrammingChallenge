51-IR-Car
=========

此部分是红外遥控小车单片机程序

所需软件环境

+ KEIL(编译软件)
+ STC_ISP(烧写软件)

所需的硬件清单

+ STC89C52RC最小系统(单片机控制核心)
+ L293D模块(电机驱动)
+ VS1838B(红外一体化接收头)
+ 小车底盘（淘宝上面有非常多的小车底盘卖，自己任选）
+ 马达：TT马达 2个
+ 轮子：2个
+ 锂电池:2200mAH 两节7.4V + 充电器（可用充电宝代替）
+ 杜邦线若干
+ 车载手机支架（有车一族，不用的车载手机支架可以利用）
